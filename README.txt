Tic-Tac-Toe

This is a Tic-Tac-Toe game, on a 3x3 grid.

In this game, you must align three pawns in a row, a column, or a diagonal with the same color.
If you do, the game stops, and the player who aligned three pawns wins.

Player 1 has the red color
Player 2 has the blue color

To start a new game, please close the current window and re-run the file.

Morpion

Ce jeu est un Morpion sur une grille de 3x3.

Le but dans ce jeu est d'aligner trois pions de la même couleur sur une ligne, colonne ou diagonale.
Le jeu s'arrête alors et le joueur qui a aligné les trois pions a gagné.

Pour recommancer une partie, quittez l'application en fermant la fenêtre et relancez la.
